# Apple Liquid Glass App

## Project Overview
The Apple Liquid Glass App is a web application designed to showcase an interactive user experience reminiscent of Apple’s minimalist aesthetic. Built with Next.js and Tailwind CSS, this application supports advanced animations using Framer Motion and UI components from Radix UI.

## Installation
To get started with the Apple Liquid Glass App, follow these steps:

1. **Clone the repository:**
   ```bash
   git clone https://github.com/yourusername/apple-liquid-glass-app.git
   cd apple-liquid-glass-app
   ```

2. **Install dependencies:**
   ```bash
   npm install
   ```

3. **Run the development server:**
   ```bash
   npm run dev
   ```

   The application will be available at `http://localhost:8000`.

## Usage
Once the development server is running, you can navigate to `http://localhost:8000` in your web browser to view the application. You can modify the source files in the `src` directory to customize the application per your needs.

## Features
- **Responsive Design:** Utilizes Tailwind CSS for a responsive layout.
- **Dynamic Routing:** Integrated Next.js capabilities for server-side rendering and API routes.
- **Animations:** Enhanced UI animations using Framer Motion.
- **Accessibility:** Components built using Radix UI ensuring accessibility standards.

## Dependencies
The project is built with the following dependencies:

- **Next.js:** `14.0.0`
- **React:** `^18`
- **React DOM:** `^18`
- **Framer Motion:** `^10.16.0`
- **Radix UI Components:**
  - Dialog: `^1.0.5`
  - Dropdown Menu: `^2.0.6`
  - Tabs: `^1.0.4`

Additionally, the project uses several development dependencies, including TypeScript, ESLint, and Tailwind CSS.
